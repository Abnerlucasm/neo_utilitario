Guia de Markdown

Formatação Básica

  • **texto** → texto em negrito
  • *texto* → texto em itálico
  • # Título → título principal
  • ## Subtítulo → subtítulo
  • - item → item de lista
  • 1. item → item numerado
  • `código` → código inline
